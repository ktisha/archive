dfs-server

Для запуска выполнте
java -jar "server.jar"

ВНИМАНИЕ: запускайте только в текущей директории

В файле centralserver.properies находятся настройки сервера.

В базе существуют пользователи
--- user --- pass ---
student      student
user1        user1
user2        user2
user3        user3
user4        user4

В файле storage.conf описаны адреса storage agent
в формате
storage_host storage_port
каждая запись об агенте должна начинаться с новой строки
